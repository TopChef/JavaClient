Detailed Description

Describes the possible HTTP status codes that can be returned by making an HTTP request from the TopChef API. These codes map onto familiar HTTP status codes in the following way

Enum Status Code
OK 200
CREATED 201
ACCEPTED 202
NO_CONTENT 204

NOT_FOUND 404
